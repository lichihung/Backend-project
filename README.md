# Backend-project

**Back-end Developer Capstone Project**

- This project is a full-stack web application built with Django and Django REST Framework (DRF) for the Little Lemon restaurant.
- It includes static web pages, a MySQL backend, and RESTful APIs for managing menu items and table bookings.
- User authentication and registration are also implemented using Djoser and DRF Token Authentication.

**Features**

- Static Website Pages served via Django templates
- Menu API for listing and managing menu items
- Booking API for creating and viewing table reservations
- MySQL Database integration
- User Registration & Authentication (Djoser + Token)
- Unit Tests for models and API endpoints
- API Testing using Insomnia or Postman

**Technologies Used**

- Python 3
- Django 5
- Django REST Framework
- MySQL
- Djoser
- Insomnia / Postman (for API testing)
- Git & GitHub (for version control)
